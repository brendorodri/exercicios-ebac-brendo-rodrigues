# hora-de-acordar

1. Abro os olhos
2. Levanto da cama
3. Faço um breve alongamento
4. Caminho até o banheiro
5. Pego minha escova de dente
6. Escovo meus dentes
7. Saio do banheiro
8. Caminho até a cozinha
9. Preparo um lanche saudável
10. Me alimento com o lanche que preparei
11. Saio da cozinha
12. Levo os cachorros para passear
